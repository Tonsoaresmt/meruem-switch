# Atualizacoes do App

O Meruem Switch pode atualizar o proprio `.nro` usando as releases publicadas
neste repositorio.

## Para Usuarios

Quando existir uma versao mais nova, o app pode mostrar um aviso ao abrir.

Se voce aceitar:

1. o app baixa o novo `Meruem.nro`;
2. substitui o arquivo atual no SD;
3. voce abre novamente usando a versao nova.

Se preferir atualizar manualmente:

1. abra https://github.com/Tonsoaresmt/meruem-switch/releases/latest;
2. baixe `Meruem.nro`;
3. copie para `sdmc:/switch/Meruem/Meruem.nro`;
4. substitua o arquivo antigo.

## Sobre Assinatura

Atualizar o app nao altera seu plano Meruem.

O app e gratuito para baixar, mas a leitura sem limite depende de assinatura
ativa na plataforma Meruem. O plano de entrada parte de **R$ 6,90 por 3 meses**
para liberar leitura sem limite na area escolhida.

Esse valor baixo ajuda a manter o projeto online e acessivel, cobrindo custos da
plataforma, armazenamento, trafego, manutencao e melhorias.

## Se Algo Der Errado

Se o atualizador nao conseguir concluir:

- baixe o `Meruem.nro` manualmente pela pagina de releases;
- copie novamente para o SD;
- substitua o arquivo antigo;
- abra o app outra vez.

Atualizacoes nao removem sua conta, favoritos ou historico salvos na plataforma
Meruem.
